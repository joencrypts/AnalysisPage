# Infrastructure Assessment Tool

A modern web application for infrastructure assessment and review, built with React, TypeScript, and Tailwind CSS.

## Features

- Drag and drop image upload
- Real-time infrastructure assessment
- Cost estimation and analysis
- Responsive design for all devices
- Modern UI with smooth animations
- Secure assessment submission

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- ESLint
- Prettier

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/infrastructure-assessment-tool.git
cd infrastructure-assessment-tool
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
# or
yarn build
```
